前端所需相关组件封装

框架：vue

插件：移动端ui框架——————————vant_ui
     npm i vant -S

     pc端ui框架———————————ant_design of vue
     npm install ant-design-vue --save

     pc端ui框架———————————element
     npm i element-ui -S

     图表—————————————————echart
     npm install echarts --save

     钉钉dingtalk
     npm install dingtalk-jsapi --save

使用方法：
    根据需求，复制相关组件文件即可，子组件标签直接v-model绑定数据
